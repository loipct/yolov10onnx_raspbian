Using yolov10 onnx on Raspberry Pi 4 with Python

##  Os & Python 

Raspbian 64 bit & 3.11.0

## Usage

1. Create pythonn env: 
```
python -m venv venv
```
2. Install Requirements:
```
source venv/bin/activate
sudo pip install -r requirements.txt
```
3. Run the detection:
```
python3 detect.py  
```
